## ⚙️ How It Works

1. User inputs numbers using buttons
2. Operator is selected (`+`, `-`, `*`, `/`)
3. Previous operation is calculated before applying a new one
4. Result is displayed when `=` is pressed
5. State resets or updates accordingly
